# react-native-tf-lite

run TensorFlow Lite models in React Native Android apps

## Installation

```sh
npm install react-native-tf-lite
```

## Usage

```js
import { multiply } from 'react-native-tf-lite';

// ...

const result = await multiply(3, 7);
```
